# **App Name**: LEDTools Suite

## Core Features:

- Screen Resolution Calculator: Calculate LED screen resolution in pixels, meters, millimeters, feet, and inches.
- Power and Data Calculator: Calculate weight, total ampere, power, and data requirements for the LED screen setup.
- Pixel Map Generator: Generate a pixel map in a checkerboard pattern to visualize LED layout.
- Wiring Diagram Creator: Create a wiring diagram for the LED screen setup.
- Raster Map Generator: Generate a raster map for the LED screen to assist content designers and media server personnel.
- AI Troubleshooting: Uses a tool to provides troubleshooting steps using generative AI based on screen configuration and parameters.
- Import / Export Project Configuration: Allows the user to import/export the configuration data in JSON format.
- LED Product Options: Tells user which LED product is close to the required dimension

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) to convey professionalism and reliability. This color is versatile, working well in a dark UI.
- Background color: Very dark blue (#0A192F), a desaturated version of the primary, creates a professional dark mode aesthetic.
- Accent color: A lighter shade of blue (#558BE2), stands out against the dark background to highlight key interactive elements.
- Body and headline font: 'Inter' (sans-serif) provides a modern and neutral look, suitable for both headlines and body text.
- Use clean and professional icons related to LED screens and technical specifications.
- Maintain a clean and organized layout with a header for navigation tabs and a left-side panel for settings. Allow users to switch between dark and light mode.
- Subtle transitions and animations when switching between tabs or generating diagrams.
